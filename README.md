# fortigotools

This is example code for learning: alter the main() call being made to see the results.

Steps:
- Install Golang
- Git clone this repo
- CD to folder
- Rename the config.yaml.example to config.yaml and change hostname and apikey per your own.
- Run "go mod tidy"
- Run "go run ."
- Have fun !
